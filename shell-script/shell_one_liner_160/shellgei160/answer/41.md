### 解答
```
$ cat annotation.md | grep -oE '\[\^[^[]+\]:?' | sed 's/]$/] 目印/' | sed 's/:$/ 注釈文/' | sort | awk '{print $2,$1}' | uniq -f1 -u
```
### 別解
```
別解（山田）$ cat annotation.md | grep -oE '\[\^.*\]' | sort | uniq -u | grep -F -f- annotation.md | grep -oE '\[\^.*\]:?' | awk -F: '/:/{print "注釈文",$1}!/:/{print "目印",$0}'
```
