### 解答
```
$ awk -F: '{print $(NF-2)}' access.log | awk '$1<"12"{print "午前"} $1>="12"{print "午後"}' | sort | uniq -c
```
### 別解
```
別解1（上田）$ cat access.log | grep -o '..:..:.. +0900' | sed 's/:.*//' | awk '{print $1<"12"?"午前":"午後"}'| sort | uniq -c
別解2（eban、上田改）$ sed -r 's@.*\[|\]|/@@g;s/:/ /' access.log | date -f- +%p | sort | uniq -c
```
